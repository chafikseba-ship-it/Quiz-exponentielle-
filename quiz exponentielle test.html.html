⚡ Les 2 règles à retenir avant tout

1. ex > 0 pour tout x réel (toujours)
2. eA = eB ⇔ A = B   |   eA < eB ⇔ A < B

Avec ces deux règles, tu fais 80% des exercices.

🎯 La dérivée : à quoi ça sert

La dérivée donne le sens de variation de la fonction.

• f '(x) > 0 sur un intervalle → f est croissante sur cet intervalle.

• f '(x) < 0 sur un intervalle → f est décroissante sur cet intervalle.

• f '(x) = 0 en un point → souvent un maximum ou un minimum.

Étudier les variations = étudier le signe de la dérivée.

🧩 Pourquoi factoriser la dérivée

Une dérivée comme f '(x) = ex + x ex : impossible de connaître son signe.

On factorise par ex : f '(x) = ex (1 + x).

Comme ex > 0, le signe de f '(x) est celui de (1+x).

Factoriser = isoler le facteur qui décide du signe.

↓ Méthodes par type de question ↓

🔍 Résoudre une équation avec exp
📌 Tu vois : "Résoudre dans ℝ l'équation..." avec des e...

Méthode :

  1. Mettre sous la forme eA = eB. Si tu vois 1, écris e0. Si tu vois e, écris e1.
  2. Écrire : "Comme exp est strictement croissante, l'équation équivaut à A = B".
  3. Résoudre A = B.
  4. Conclure : S = {...}.
Exemple : ex+2 = e5 ⇔ x+2 = 5 ⇔ x = 3. Donc S = {3}.
⚠️ Si tu obtiens ex = nombre négatif ou zéro → aucune solution (car ex > 0).
📊 Résoudre une inéquation avec exp
📌 Tu vois : "Résoudre dans ℝ l'inéquation..." avec <, >, ≤, ≥

Méthode :

  1. Mettre sous la forme eA < eB (ou avec l'autre signe).
  2. Écrire : "Comme exp est strictement croissante, l'inéquation équivaut à A < B". Garder le même sens.
  3. Résoudre l'inéquation classique.
  4. Conclure avec un intervalle : S = ]..., ...[.
Exemple : e3x < ex+6 ⇔ 3x < x+6 ⇔ 2x < 6 ⇔ x < 3. Donc S = ]−∞ ; 3[.
⚠️ Le sens de l'inégalité ne change JAMAIS avec exp (parce qu'elle est croissante).
🧮 Simplifier une expression
📌 Tu vois : "Simplifier", "Écrire sous la forme ek", "Calculer A ="

Méthode : applique les 4 règles

  1. Multiplication : ea × eb = ea+b (additionner les exposants).
  2. Division : ea / eb = ea−b (soustraire les exposants).
  3. Puissance : (ea)n = en×a (multiplier les exposants).
  4. Inverse : 1 / ea = e−a.
Exemple : A = (e5 × e−3) / e−2 = e5−3 / e−2 = e2 / e−2 = e2−(−2) = e4.
✏️ Dériver une fonction avec exp
📌 Tu vois : "Calculer f '(x)", "Dériver f", "Montrer que f '(x) = ..."

Identifie la forme, applique la formule :

  1. Forme ex seul → (ex)' = ex.
  2. Forme eax+b → dérivée = a × eax+b. Multiplier par le coefficient devant x.
  3. Forme k × e... → la constante k reste, on dérive l'exponentielle.
  4. Forme u(x) × ex (produit) → formule (uv)' = u'v + uv'.
  5. Somme → on dérive terme à terme.

Après un produit, TOUJOURS factoriser par ex.

Exemple : f(x) = (x−1) ex Produit avec u = x−1, v = ex.
f '(x) = 1 × ex + (x−1) × ex
Factorisation : f '(x) = ex(1 + x − 1) = x ex.
🔗 Factoriser par ex
📌 Tu vois : "Factoriser", "Montrer que f '(x) = ..." (souvent après une dérivation)

Méthode :

  1. Repérer le facteur e... commun à tous les termes.
  2. Le mettre devant une parenthèse.
  3. Dans la parenthèse, écrire ce qui reste pour chaque terme.
  4. Simplifier la parenthèse.
Exemple : ex + x ex = ex × 1 + ex × x = ex(1 + x).

💡 But : pouvoir étudier le signe ensuite (puisque ex > 0).

➕ Étudier le signe d'une expression
📌 Tu vois : "Étudier le signe de", "Déterminer le signe de"

Méthode :

  1. Factoriser au maximum (souvent par e...).
  2. Écrire : "Pour tout x réel, e... > 0".
  3. Donc le signe ne dépend que de l'autre facteur.
  4. Étudier ce facteur (résoudre = 0, puis tableau de signes).
Exemple : signe de x ex ex > 0, donc signe de x ex = signe de x.
Négatif si x < 0, nul si x = 0, positif si x > 0.
📈 Étudier les variations / Dresser le tableau de variations
📌 Tu vois : "Étudier les variations", "Dresser le tableau de variations", "Sens de variation"

Méthode (toujours la même) :

  1. Calculer f '(x).
  2. Factoriser f '(x).
  3. Étudier le signe de f '(x).
  4. En déduire : positif → croissante, négatif → décroissante.
  5. Dresser le tableau de variations (ligne x, ligne signe de f ', ligne variations de f).
  6. Calculer les valeurs de f aux points importants (où f ' = 0).
Exemple : f(x) = (x−1) ex f '(x) = x ex (déjà factorisé).
Signe : celui de x.
f décroît sur ]−∞ ; 0], croît sur [0 ; +∞[.
f(0) = −1 (minimum).
🏔️ Déterminer un maximum / un minimum
📌 Tu vois : "Déterminer le minimum/maximum de", "En déduire la valeur minimale"

Méthode :

  1. Étudier les variations (donc dériver, factoriser, signe).
  2. Faire le tableau de variations.
  3. Repérer le point où f '(x) = 0 et où la fonction change de sens.
  4. Calculer f à ce point : c'est la valeur du min/max.
  5. Rédiger : "Le minimum de f sur ℝ est f(a) = ..., atteint en x = a".
📐 Démontrer qu'une suite est géométrique
📌 Tu vois : "Montrer que (un) est géométrique", "Démontrer que la suite est géométrique"

Méthode :

  1. Calculer un+1 en remplaçant n par n+1 dans la formule.
  2. Transformer pour faire apparaître un multiplié par un nombre fixe.
  3. Conclure : "un+1 = un × q donc (un) est géométrique de raison q".
  4. Préciser le premier terme.
Exemple : un = e0,5n un+1 = e0,5(n+1) = e0,5n+0,5 = e0,5n × e0,5 = un × e0,5.
Donc géométrique de raison e0,5, u0 = 1.
📏 Donner l'équation d'une tangente
📌 Tu vois : "Équation de la tangente à la courbe au point d'abscisse a"

Formule : y = f '(a)(x − a) + f(a)

Méthode :

  1. Calculer f(a).
  2. Calculer f '(x), puis f '(a).
  3. Remplacer dans la formule.
  4. Développer pour obtenir y = mx + p.
Exemple : tangente à exp en x = 0 f(0) = 1, f '(0) = e0 = 1.
y = 1 × (x − 0) + 1 = x + 1.
📊 Reconnaître croissance / décroissance exponentielle
📌 Tu vois : "Modéliser par...", une suite ou fonction de la forme ekt

Règle :

  1. f(t) = ekt avec k > 0 → croissance exponentielle (la fonction monte).
  2. f(t) = e−kt avec k > 0 → décroissance exponentielle (la fonction descend).
  3. Plus |k| est grand, plus c'est rapide.

Exemples concrets :

• Bactéries qui se multiplient → ekt

• Désintégration radioactive → e−kt

• Refroidissement d'un objet → e−kt
